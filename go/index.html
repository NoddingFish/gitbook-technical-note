Go

Go

Go 是一个开源的编程语言，它能让构造简单、可靠且高效的软件变得容易。

第一个 Go 程序 hello.go ：

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

执行以上代码输出：

$ go run hello.go 
Hello, World!
